false reserved word

The false value is one of the two immutable boolean values. The false value is falsy.

The false value can be produced by a relational operator.

false literal

The false literal is a convenient way to access the false value.

var found = false;

falsy

Every value is either truthy or falsy. These are the only values that are falsy:

  • "" empty string
  • 0 zero
  • false
  • NaN not a number
  • null
  • undefined

All other values, including empty objects and empty arrays, are truthy.

A falsy value can have consequences:

  • in the condition part of an if statement, the else branch is taken.
  • in the condition part of a loop statement, break out of the loop.
  • in the first operand of the && infix operator and, the result is the falsy value of the first operand.
  • in the first operand of the || infix operator or, the result is the value of the second operand.
  • in the first operand of the ? ternery operator condition, the result is the value of the third operand.
  • in bitwise operators, a falsy operand is treated the same as 0.
